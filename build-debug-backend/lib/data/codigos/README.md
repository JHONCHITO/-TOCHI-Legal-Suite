# Codigos Legales Colombianos

## Estructura de Archivos

Cada codigo tiene su propio archivo TypeScript. Para agregar articulos:

1. Abre el archivo del codigo correspondiente
2. Agrega los articulos en el array `articulos`
3. Guarda el archivo

## Formato de cada articulo:

```typescript
{
  numero: "1",           // Numero del articulo
  titulo: "Titulo",      // Titulo opcional
  contenido: "Texto completo del articulo...",
  libro: "Libro I",      // Opcional
  titulo_libro: "De las personas",  // Opcional
  capitulo: "Capitulo I" // Opcional
}
```
